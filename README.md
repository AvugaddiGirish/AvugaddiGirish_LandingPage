# GIRISH — Interactive AI Automation Portfolio

A high-performance, interactive 2D vector portfolio designed for an AI Automation Engineer. Built entirely with vanilla web technologies, this portfolio features a generative background, Web Audio API integration, and complex SVG micro-interactions.

## ✨ Key Features
* **Generative Canvas Background:** A custom hardware-accelerated particle system using the `Path2D` API to render drifting tech vectors (Python logo, microchips, n8n nodes, etc.).
* **Dynamic SVG Micro-Interactions:** CSS-driven hover states including wheel rotation, wind effects, and data packet routing.
* **Web Audio API Engine:** Synthesized, real-time audio feedback including F1 engine revs, an aura hum, and mechanical keyboard/controller sequences.
* **Responsive & Accessible:** Fully optimized for mobile touchscreens with built-in battery/CPU management using the Page Visibility API. Includes full keyboard navigation (`Tab`, `Enter`, and hidden `1-4` hotkeys).

## 🛠 Tech Stack
* **HTML5:** Semantic structure and inline SVG drawing.
* **CSS3:** Advanced keyframe animations, CSS variables, and glassmorphism.
* **Vanilla JavaScript (ES6+):** Mouse parallax physics, procedural audio generation, and canvas rendering.

## 💻 Local Setup
No build tools or frameworks required. Simply download the code and open `index.html` in any modern web browser.
